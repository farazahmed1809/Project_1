# NOVA//SYS — Hardware Catalog

A sleek, responsive hardware catalog interface presenting technical products alongside their performance statistics and raw datasheets. Built with semantic HTML5, CSS3 layout paradigms, and vanilla JavaScript, this project features client-side product filtering, a side-by-side product comparison engine, and a dual-theme switch mechanism.
